# Memory game

It`s game created by VUE & TS

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Tech Stack

- Vue
- Pinia
- Vue router
- Type Script
- CSS
- HTML


### Deploy
[Memory-game](https://visheyt-memory-game.netlify.app/)
